ПОЗИТИВНЫЕ ТЕСТ-КЕЙСЫ
ТК 1. Поиск билетов между городами для 1 взрослого
    1. Открыть главную страницу Aviasales
    2. Ввести город в поле "Откуда"
    3. Ввести город в поле "Куда"
    4. Выбрать дату вылета в будущем
    5. Выбрать "Обратный билет не нужен" в дропдауне Обратно
    6. Нажать на "Найти билеты"

    Ожидаемый результат:
        Отображается список доступных билетов с корректной информацией:
        - города отправления и прибытия совпадают с введенными
        - дата вылета совпадает с выбранной


ТК 2. Сортировка найденных билетов по цене от дешевых к дорогим
    1. Выполнить поиск билетов (TК 1)
    2. Нажать на дропдаун Сортировка
    3. Выбрать сначала дешевые

    Ожидаемый результат:
        Отображается список билетов, отсортированных по возрастанию цены


ТК 3. Просмотр информации о билете
   1. Выполнить поиск билетов (TК 1)
   2. Выбрать билет

   Ожидаемый результат:
        Информация в билете совпадает с информацией на превью билета:
            - города отправления и прибытия;
            - дата и время отправления и прибытия;


НЕГАТИВНЫЕ ТЕСТ-КЕЙСЫ
ТК 4. Поиск билетов без пункта назначения
   1. Открыть главную страницу Aviasales
   2. Ввести город в поле "Откуда"
   3. Выбрать дату вылета в будущем
   4. Нажать на "Найти билеты"

   Ожидаемый результат:
        Отображается сообщение об ошибке "Укажите город прибытия"


ТК 5. Поиск билетов с несуществующим пунктом назначения
   1. Открыть главную страницу Aviasales
   2. Ввести город в поле "Откуда"
   3. Ввести в поле "КУда" последовательность символов (например ааа)

   Ожидаемый результат:
        В дропдауне отображается текст "Ничего не нашли. Проверьте, правильно ли вы написали название"
